PROJECTS WE CREATED

Here are some of the projects my groupmates and I worked on during our first and second year as IT students.

T.H.I.S APPOINTMENT SYSTEM

School Year 2024-2025

T.H.I.S (Tooth Health and Ideal Smile) Appointment System is a project requirement for the subject Computer Programming 2. This system was developed to help dental clinics manage and monitor patient appointments and records in a more organized and efficient way.

The system focuses on simplifying appointment tracking by allowing clinic staff to manage patient information, appointment schedules and appointment statuses such as completed visits, cancellations, no-shows and pending appointments. It also helps monitor the clinic's appointment activities by providing organized records that can be reviewed over different periods.

Documentation & UI Designer Visual Studio • C# Database

EVENT REGISTRATION FORM

School Year 2025

This project is a web-based Event Registration Form created as part of our HTML and CSS activity. The goal of this project is to practice designing and styling different form elements while creating an organized and user-friendly registration interface.

The form includes various input features such as personal information fields, gender selection, event package options, interests through checkboxes, message section and submission controls. This project focuses on improving our understanding of form structures, input types and CSS styling to create a clean and responsive web design.

Frontend Developer HTML CSS

AUTOMATIC ROOM LIGHT SYSTEM

School Year 2026

This is an IoT-based project that we created as a requirement for the subject Application Development and Emerging Technologies during our 2nd Year. This project was developed by our team of 8 members where we explored the use of Arduino and hardware components to create an automated room system.

The purpose of this project is to create a smart room setup where an LED light automatically turns on when motion is detected, while the LCD displays the current system status. It also includes an automatic window-opening feature using a motor.

Project Initiator ARDUINO
